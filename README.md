# FOSSBilling-OLSPanel Server Module
This repository is for the OLSPanel server manager for FOSSBilling.

This server manager has been re-written using the RESTful API.  This is a community developed and maintained package.

## Usages
1. Creates new user + domain
2. Suspend, Unsuspend User
3. Change Password
4. Cancel/Delete a user


## Installation
1. Download the `OLSPanel.php` file from the source code
2. Copy the downloaded `OLSPanel.php` file to `/library/server/manager/OLSPanel.php`in your FOSSBilling instance.

## Custom Package Values (Required)
- pkg_id
- php_version
